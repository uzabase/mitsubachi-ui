## ロゴ・アイコンについて

### ロゴ
ユーザベースおよび各サービスのロゴの著作権は株式会社ユーザベースに帰属します。
ロゴの使用ルールについては、[会社概要ページ > メディアキット](https://www.uzabase.com/jp/company/#media-kit)からダウンロードできるメディアキットに含まれるガイドラインを確認してください。

### アイコン
本プロジェクトに含まれるアイコンの著作権は株式会社ユーザベースに帰属します。

## Model Context Protocol (MCP) server
非公式のModel Context Protocol server, [mitsubachi-ui-mcp](https://github.com/uzabase/mitsubachi-ui-mcp)があります。
